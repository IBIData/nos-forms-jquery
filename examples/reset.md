#Type
##reset

By default, the reset button will reset your form.

####Example with all options

```json
{
    "name": "reset",
    "type": "reset",
    "classname": "btn btn-danger",
    "formGroup": true,
    "data": {
        "field": 0,
        "testing": 123,
    },
    "inline": true,
    "title": "",
    "value": "Reset"
}
```
