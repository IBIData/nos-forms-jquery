#Property
##data

Will add data- tags to your elements.

####Single data attributes

```javascript
"data": {
    "test": 1   // will render 'data-test="1"' on your element
}   
```

####Multiple data attributes

```javascript
"data": {
    "test": 1,
    "anothertest": 2   // will render 'data-test="1" data-anothertest="2"' on your element
}  
```
